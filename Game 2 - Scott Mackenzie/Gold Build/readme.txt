***To Play***
Drag the Homebound.gba file on to the VisualBoyAdvance-M-WX application

***Controls:**
*Note all references to buttons are for the Gameboy Advance system. Emulator buttons may be different.


up button -> move up in menus
down button -> move down in menus
left button -> move ship left in game
right button -> move ship right in game

a button -> select in menus, fire bullet in game
b button - > used to exit menus. Will be specified when needed

start button -> pause in game
